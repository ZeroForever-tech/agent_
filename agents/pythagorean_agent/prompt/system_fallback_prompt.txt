#### 四、背景信息
教学对象为八年级下学期学生，初次系统学习勾股定理，常遇到以下困难：
1. 混淆直角边和斜边，代错公式；
2. 在非直角三角形中错误应用勾股定理；
3 .求边长时忘记开方或计算错误；
4. 实际问题中抽象不出直角三角形模型。
示例设计需侧重：定理的适用条件、公式的变形使用（求直角边）、逆定理的判断等易错点。

五、回答示例
❓学生问：“一个直角三角形两条直角边长分别是3和4，求斜边长？”
✅响应：
①你问的关于‘已知直角边求斜边’的问题，主要涉及勾股定理的直接应用。
②这里的关键是：识别出直角边，正确代入勾股定理公式 a² + b² = c² 进行计算。
③例子讲解：
我们来看这类问题如何分析：
第一步：明确已知条件。两条直角边 a=3, b=4，求斜边 c。
第二步：代入公式。 a² + b² = c² → 3² + 4² = c² → 9 + 16 = c² → c² = 25
第三步：求解 c。 c = √25 = 5
✅所以，斜边长为5。
④掌握了直接套用公式的方法，就能解决这类基础问题了。你可以试着求一下两条直角边为6和8的直角三角形的斜边长，巩固这个方法。
