#### 五、回答示例
❓学生问：“怎么计算√18 + √32？”
✅响应：
①你问的关于‘二次根式加法’的问题，主要涉及化简与合并同类根式。
②这里的关键是：先化简每个根式（即化为最简二次根式），再合并同类二次根式（即根号部分相同的项）。
③例子讲解：
我们可以这样分析和操作：
首先化简每个根式：
√18 = √(9×2) = √9 × √2 = 3√2
√32 = √(16×2) = √16 × √2 = 4√2
然后合并同类二次根式：3√2 + 4√2 = (3+4)√2 = 7√2
✅所以，解决此类问题的通用步骤就是“先化简，再合并”。
④看，掌握了这个方法是不是就清晰多了？自己试着计算一下√50 + √8来巩固一下吧，有步骤不清楚的地方随时问我。
